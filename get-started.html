Get Started

Don't know how to setup the bot? Here's how!

hackerman14 is a bot which is still in development, but getting better!

Invite the Bot
To invite the bot, here are the steps:
  1. Click the "Invite" button at the top of the page.
    It will take you to the Discord bot invite page.
  2. Select the options from the page.
    Select the server that you want the bot to join, and give it the preset permission that I've given (you can skip this if you want a custom permission setup).
  3. Click the "Authorize" button.
    The button looks shiny and doesn't seem spoofy...
  4. Do the reCAPTCHA.
    Google claimed that robots aren't able to do reCAPTCHAs, but somehow a robot did...
  5. Done, close the Authorize tab and check your server
    That's it! You've successfully invited the bot!

Interact with the Bot
Just simply type "rh!help" and all the commands will show up!
